# 🚌 Bus Ticket Booking System (Java)

A console-based Bus Ticket Booking System developed using Java and MySQL.

## 📌 Features
- User Registration & Login
- View Available Buses
- Book Tickets
- Cancel Booking
- Update Booking
- Admin Panel

## 🛠 Tech Stack
- Java
- JDBC
- MySQL
- OOP Concepts

## 🚀 How to Run
1. Clone the repository
2. Configure MySQL database
3. Run Main.java
